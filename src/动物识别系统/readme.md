## 简介

> 基于TensorFlow2.3 支持GPU，环境已配置好，笔记本4050显卡训练6次仅用2分钟，模型保存为h5文件，
- 可识别的对象有狗,马,大象,蝴蝶,母鸡,猫,牛,羊,蜘蛛，松鼠
- 模型使用DenseNet201，可调整训练次数，
- 已经训练好的模型准确度百分之八十以上
- 大量注释
- 自带训练数据集，每种类别数量在两千左右，全部图片数量为26332张
